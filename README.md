# Impact Article: Builds 1–3 Starter Template

This is your starting point for Builds 1–3. You'll work on this same project across all three builds, progressively enhancing it.

## What's Included

- **index.html** — Basic HTML5 boilerplate
- **style.css** — Minimal reset and base styles

## The Arc

- **Build 1:** Build the HTML structure and CSS design
- **Build 2:** Make it responsive and accessible
- **Build 3:** Add JavaScript interactivity

## Resources

The [course website resources](https://webdev.wat.tech/resources/) have everything you need: HTML and CSS reference guides, commit message guidance, and code commenting guidance.
